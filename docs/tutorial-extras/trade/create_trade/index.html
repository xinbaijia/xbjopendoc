创建订单

openapi/param2/1/com.xinbaijia.trade/xinbaijia.trade.createTrade/{apikey}

系统级输入参数​

名称类型是否必须描述
_xop_timestampint是请求毫秒时间戳
_aop_signaturestring是请求签名(签名规则
access_tokenstring是用户授权令牌

应用级输入参数​

  • address: dict 收件人地址

    名称类型是否必须描述示例值
    namestring是收件人姓名张三
    mobilestring是收件人电话13167819879
    provincestring是省份广西壮族自治区
    citystring是城市南宁
    areastring是地区兴宁区
    addressstring是详细地址五塘镇兴东街道1505号
    poststring否邮政编码100000

  • is_custom: true | false 是否是定制

  • products: list 商品列表

    名称类型是否必须描述示例值
    quantityint是商品数量5
    product_idint是商品id(对应1688的商品ID)771084988960
    specsstring[]是商品规格["7号带袜", "16码"]
    customdict[]取决于is_custom定制参数[{"name": "name", "value": "FÜLLKRUG"}]

    • custom 定制参数说明

      定制参数格式为:

      定制姓名：{"name": "name", "value": "姓名"}

      定制号码：{"name": "number", "value": "17"}
